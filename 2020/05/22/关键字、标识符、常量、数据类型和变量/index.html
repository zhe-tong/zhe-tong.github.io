关键字、标识符、常量、数据类型和变量

关键字：

有特殊含义的、被保留的、不能随意使用的字符。

特征：

  1. 完全小写的字母。
  2. 在增强版记事本中（如notepad++中）有特殊颜色。

标识符：

标识符指自己定义的内容，如类名、方法名和变量名

命名规则：

  1. 只能包含大小写字母、数字、$和下划线
  2. 不能数字开头
  3. 不能是关键字

命名规范：

  • 类名：首字母大写，后面每个单词首字母大写（也称大驼峰式），如HelloWorld
  • 变量名：首字母小写，后面每个单词首字母大写
  • 方法名：同变量名

常量：

在程序运行期间，固定不变的量。

常量分类：

  1. 字符串常量：凡是用双引号引起来的部分，叫做字符串常量，例：”abc”
  2. 整数常量：直接写上的数字，没有小数点，例：100
  3. 浮点数常量：直接写上的数字，有小数点，例：2.2
  4. 字符常量：凡是用单引号引起来的单个字符，只能是单个字符，例：’A’
  5. 布尔常量：ture、false
  6. 空常量：null 代表没有任何数据

基本数据类型：

  • 整数型 byte short int long
  • 浮点型 float double
  • 字符型 char
  • 布尔型 boolean

变量：

程序运行期间，内容可以发生改变的量

创建一个变量并使用的格式：

数据类型 变量名称；//创建了一个变量

变量名称 = 数据值；//赋值，将右边的数据值交给左边的变量
例：

    int num1;

    num1 = 100;

一步到位的格式：

数据类型 变量名称 = 数据值；//创建一个变量的同时赋值
例： 

    int num1 = 100;

变量的注意事项：

  1. 如果创建多个变量，变量之间名称不能重复
  2. 对于float和long来说，不能省略F和L字母后缀
  3. 如果使用byte或者short，那要注意右侧数据的赋值范围
  4. 没有赋值的变量不能直接使用
  5. 变量使用不能超过作用域范围。【作用域】：从定义变量的一行直至所属大括号结束的范围
  6. 可以通过一个语句创建多个变量，但建议不要这么写
