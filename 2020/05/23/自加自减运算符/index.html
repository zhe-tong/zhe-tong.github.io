自加自减运算符

自加自减运算符

自增运算符：++

自减运算符：–

含义：

让一个变量涨/降一个数字1

使用格式：

写在变量前或后，例i++或++i

使用方法：

  • 单独使用：不和其它操作混合，自己独立为一个步骤
    例： 

      int num1 = 10;
  System.out.println(num1);
  ++num1;
  System.out.println(num1);

  • 混合使用：和其他操作混合，如赋值、打印等

使用区别：

  1. 在单独使用时，前++和后++没有任何区别
  2. 混合使用时有重大区别

    • 如果是【++i】，那么【马上加1】，再拿着结果使用。 【先加后用】

    • 如果是【i++】，那么先使用原来的数值，【再让变量加1】。 【先用后加】
      例： 

            //与打印操作混合
      int num2 = 20;//前++，先加后用
      System.out.println(++num2);//结果21
      System.out.println(num2);//结果21

      int num3 = 20;//后++，先用后加
      System.out.println(num3++);//结果20
      System.out.println(num3);//结果21

注意事项：

只有变量可以使用自增自减运算符，常量不可以。
