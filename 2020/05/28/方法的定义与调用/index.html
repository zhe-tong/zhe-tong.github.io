方法的定义与调用

方法的定义与调用

定义方法的完整格式：

修饰符 返回值类型 方法名称(参数类型 参数名称,...){
    方法体
    return 返回值;
}

方法其实就是若干语句的功能集合。举个例子，方法好比工厂。

蒙牛工厂   原料：奶牛、饲料、水
         产出物：奶制品

* 参数（原料）：进入方法的数据
* 返回值（产出物）：从方法出来的数据
  • 修饰符：目前对于新手的我来讲，public static可以满足我大部分编程需求。
  • 返回值类型：方法产生的数据结果是什么类型。
  • 方法名称：命名规则为小驼峰式。
  • 参数类型：进入方法的数据是什么类型。
  • 参数名称：进入方法的数据对应变量的名称。
    • ps：参数如果有多个，使用逗号“,”进行分割。
  • 方法体：方法需要做的事，若干行代码。
  • return：两个作用，第一停止代码，第二将后面的返回值还给调用处。
  • 返回值：也就是方法执行后最终产生的数据结果。

调用格式：

方法的三种调用格式。
    1. 单独调用：方法名称(参数);
    2. 打印调用：System.out.println(方法名称(参数));
    3. 赋值调用：数据类型 变量名称 = 方法名称(参数);

方法有参数与无参数的区别

  • 有参数：小括号当中有内容，当一个方法需要一些数据条件才能完成任务的时候就是有参数。
    • 例如两个数字相加，必须知道两数字各是多少才能相加。
  • 无参数：小括号留空。一个方法不需要任何数据条件自己就能独立完成任务就是无参数。
    • 例如定义一个方法，打印固定十次的Helloworld。

使用方法时的注意事项：

  1. 方法应该定义在类中，但是不能再方法当中再定义方法，不能嵌套。
  2. 方法定义前后顺序无所谓。
  3. 方法定义之后不会执行，如果希望执行，一定要调用。
  4. 如果方法有返回值，必须写上“return 返回值；”，不能没有
  5. return后面的返回值数据，必须和方法的返回值类型对应。
  6. 对于一个void没有返回值的方法，格式只能写return不能带有后面的返回值。
  7. 对于方法当中最后一行的return可以省略不写（如果没有返回值）。
  8. 一个方法中可以有多个return语句，但同时只有一个会被执行到，两个return不能连写。
