关键字private的使用方法

关键字private

面向对象三大特征：封装、继承、多态。
这是之前所学到的概念，其中封装性在Java中的具体体现：

  1. 方法是一种封装
  2. 关键字private也是一种封装

封装就是将一些细节信息隐藏起来，对于外界不可见。

private关键字用于以下情况：

定义变量的数值时，无法阻止不合理数值被设置。这时用private关键字将需要保护的成员变量进行修饰，如下代码：

    private int age;

但一旦使用private进行修饰，那么仅本类当中可以随意访问，超出了本类范围之外就不能再直接访问了。

这时可以采用间接访问private成员变量的方法，就是定义一对getter/setter方法。

//这个成员方法专门用于向希望进行设定的变量设置数据
//规则：set+首字母大写变量名。
//必须有参数无返回
public void setAge(int num){
    if (num >= 0 && num < 100) {
        age = num;
    }else {
        System.out.println("数据不合理！");
    }
}

//这个成员方法专门用于获取age的数据
//规则：get+首字母大写变量名。
//必须返回无参数
public int getAge(){
    return age;
}

对于基本类型当中的boolean值，他的getter方法一定要写成isXxx的形式，而setter规则不变。例：

public boolean isMale(){
    return male;
}

然后在调用时直接根据需求调用getter/setter方法即可，例：

    stu.setName("tz");
    stu.setAge(30);
    stu.setMale(true);

小结

简单来说，private关键字的功能就是根据你的需要对输入的数据加以限制，从而阻止不合理的数据的输入，起到提高代码安全性的作用。
