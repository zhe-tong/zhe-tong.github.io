Basic Linux Commands 及 9311数据库lab的环境搭建

lab中linux操作系统的基础：

以下先记录一些常用的linux commands

  1. “pwd” 指 “print working directory”, 也就是打印当前路径
    • “cd ../“ cd指 “change directory”, 切换当前目录至父目录
    • “cd ./“, 切换至当前目录
    • “cd ~” 指 “home directory”
    • “cd /“ /指 “absolute path”, 切换至指定路径
    • “ls” 指 “list directory”, 查看当前目录
    • “ls -l” 指 “long listing”, 以长数据串的形式列出当前目录
    • “ls -a” 指 “show all the hiden files and current dirctory”, 列出全部文件(包括.开头的隐藏文件)
    • “ls -la” 可获得更多detail
    • “cp” 指 “copy”, 复制操作
    • “cp -r” 指 “recursive copy”, 递归式复制，用于目录的复制
  5. “rm” 指 “remove”, 删除操作
  6. “cat” 指 “display the content”, 查看文件内容
  7. “mv” 指 “move files and directories”, 移动文件
  8. “ssh” 指 “securely connect to remote server”,远程服务

作为操作Linux系统的基础，熟练掌握命令行在初学阶段显得尤为重要，可以多做练习。同时随着学习，更多的操作命令也会在之后更新。

lab的环境搭建

  • 首先登陆网址http://vlabgateway.cse.unsw.edu.au
    点击1024*768
    输入zID及密码登陆
  • 右键打开terminal
  1. 创建虚拟机（Creating a Virtual Host）

    1. 输入

       ssh grieg

      登陆grieg

    2. 输入

       hostname

      检查是否成功登陆进入grieg

    3. 输入

       priv srvr

      创建一个新的目录来存放你的服务器文件
      可以输入

       cd /srvr/zID

      检查该目录是否创建成功

  2. 设置postgreSQL服务器（Setting up your PostgreSQL Server）

    1. 输入

       cs9311/bin/pginit

      该命令将在/srvr/zID/目录下创建一个名为pgsql的子目录，同时放置一个名为env的文件

    2. 输入

       source /srvr/YOU/env

      激活env中的环境设置

    3. 修改文件.bash_profile或者.bashrc或者.profile

       vim .profile

      接着在文件中输入

       if [ `hostname` = "grieg" ]
 then
     priv srvr（前面留4个空格）
     source /srvr/zID/env
 fi

      这里空格的要求很严格，不能出错。键入结束后:wq保存退出

    4. 输入

       source /srvr/zID/env

      完成这些步骤后之后的登陆便可以跳过这些重复的命令

  3. 使用PostgreSQL服务器（Using your PostgreSQL Server）

    1. 输入

       ~cs9311/bin/pgs

      此为控制pgs的命令

    2. 然后就可以用

       pgs start
 pgs status
 pgs stop
 psql MyDatabase
 psql -l

      对服务器进行控制
